Política de Privacidade

Última atualização: Janeiro de 2026. Esta política descreve como o CurrículoEU recolhe, usa e protege seus dados pessoais, em conformidade com o Regulamento Geral de Proteção de Dados (RGPD/GDPR).

Quais dados recolhemos

Para gerar seu currículo, recolhemos temporariamente: nome, e-mail, telefone, informações de experiência profissional e formação. Esses dados são usados exclusivamente para gerar o documento solicitado.

Como usamos seus dados

  • Os dados são processados para gerar o currículo via IA.
  • Não armazenamos seus dados pessoais após a geração do currículo.
  • Não vendemos, partilhamos ou cedemos seus dados a terceiros para fins comerciais.

Cookies e análise

Podemos usar cookies técnicos essenciais para o funcionamento do site. Nenhum dado é usado para publicidade direcionada.

Seus direitos (GDPR)

Tem direito a acesso, retificação, eliminação e portabilidade dos seus dados. Para exercer esses direitos, contacte: suporte@curriculos.eu

Contacto

Para qualquer questão relacionada com privacidade: suporte@curriculos.eu

Termos de Uso

Última atualização: Janeiro de 2026. Ao usar o CurrículoEU, você concorda com os termos abaixo.

Uso do serviço

O CurrículoEU fornece uma ferramenta de geração de currículos assistida por IA. Os currículos gerados são sugestões e devem ser revisados pelo usuário antes de serem enviados a empregadores.

Plano Gratuito

O plano gratuito permite a geração de 1 currículo por sessão, com marca d'água. Para uso ilimitado, é necessário o Plano Premium.

Plano Premium

A subscrição é gerida pelo Lemon Squeezy. Pode cancelar a qualquer momento diretamente pela plataforma, sem qualquer custo adicional. O cancelamento tem efeito no final do período já pago.

Responsabilidade

O CurrículoEU não garante a obtenção de emprego. O serviço fornece uma ferramenta de apoio à candidatura. Os resultados dependem de múltiplos fatores alheios ao serviço.
